# Sobre

## Projeto

Fiscal Renderer é uma reescrita nativa em Go para gerar documentos fiscais
auxiliares brasileiros em PDF a partir de XML. O repositório mantém o código
Python original, fixtures e PDFs gerados como material de referência enquanto a
biblioteca e o CLI em Go preservam a paridade.

## Créditos

O projeto upstream é um fork do
[nfe_utils](https://github.com/edsonbernar/nfe_utils), originalmente criado por
[Edson Bernardino](https://github.com/edsonbernar).
